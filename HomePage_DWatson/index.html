About CIS 3344

This course focuses on current technologies in client side web scripting, such as: HTML, CSS, responsive web design techniques, JavaScript, JavaScript libraries (such as jQuery), JSON (or other web data formats), Ajax (a technique using JavaScript to make asynchronous calls to web APIs), and client side web application frameworks (such as AngularJS). Students create web applications that access a database using client side code that invokes web APIs.Course Description
